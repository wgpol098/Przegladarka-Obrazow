# Przeglądarka obrazów

This is a simple image viewer. The browser has many possibilities, you can apply filter types to photos, search only photos with faces and slide show.

<h2>Functionality</h2>
<ul><li>Face detection</li>
  <li>Displaying photos of faces</li>
  <li>Detecting text in the photo</li>
  <li>Histogram preview</li>
  <li>selection of photo folders</li>
  <li>Slideshow</li>
  <li>Applying Filters to the photo</li>
  <li>Negative filter</li>
  <li>Binarization filter</li>
  <li>Grayscale filter</li>
  <li>Rotate channels</li>
  <li>Histogram alignment</li>
  <li>Histogram stretching</li>
  <li>etc.</li>
  </ul>
